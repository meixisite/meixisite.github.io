1.机器阅读理解

机器阅读理解包括 4 种比较常见的任务：完形填空，多项选择，片段抽取，自由作答。

  • 完形填空：在文章中隐藏一些单词，然后预测被隐藏的单词。

  • 多项选择：给定一个问题、一段背景文字和一些候选答案，让模型判断哪些是正确答案。

  • 片段抽取：给定一个问题、一段背景文字，从文本中抽取出一些单词，作为答案。

  • 自由作答：给定一个问题、一段背景文字，生成单词序列作为答案。

本文介绍的 Gated Convolutional Networks 主要用于多项选择问题，如 SemEval-2018 Machine Comprehension Using Commonsense Knowledge 任务，常识阅读理解题。

该任务数据集的每一个样本都包含 (P, Q, A, y)，P 表示一段文字，Q 表示一个问题，A 表示一个回答，y 表示该回答是不是正确的回答。

数据集 P、Q、A、y

中山大学的团队提出了一种 Gated Convolutional Networks 用于处理这样的问题，模型主要包括输入层、卷积层和输出层。其中的核心是卷积层用到的 Gated Convolution (门卷积)。由于 SemEval-2018 任务可以使用外部信息，例如一些常识知识，因此作者也利用 ConceptNet (一个多语言常识知识图谱) 获取单词之间的关系，并学习关系的 Embedding。

模型结构

2.1 输入层

模型的包括三个部分的输入，Question、Answer 和 Passage。每部分的输入都是类似的，由四种 Embedding 组成，下面以 Passage 的输入为例，介绍四种 Embedding。对于 Passage 中的第 i 个单词 Pi，四种 Embedding 分别如下：

  • Word Embedding：使用 300 维的 Glove 词向量，作为 Pi 的 Word Embedding。

  • Relation Embedding：关系的 Embedding 维度是 10，随机初始化。对于单词 Pi，使用 ConceptNet 找出 Pi 和 Answer 或 Question 中单词的关系 (如 UsedFor 或 HasProperty)，如果有多个关系则随机选择一个。

  • POS、NER Embedding：作者把单词归为 49 种词性和 18 种命名实体，词性的向量维度是 10，命名实体向量维度是 8。

  • Feature Embedding：主要是一些人工创造的特征，如 log 词频，word occurrence (判断 Pi 是否在 Answer 或 Question 中出现)。
