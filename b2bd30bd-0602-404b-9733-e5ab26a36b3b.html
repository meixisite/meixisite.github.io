长截图、清爽版百度、最快 16 倍速视频播放……

上期推荐中我们给大家介绍了 15 款 Chrome 扩展，它们可以大幅提高工作学习效率。这次我们新增加了一些内容，向大家推荐更多好用的 Chrome 扩展，包括一款精致的新标签页、多款效率工具、以及一个不止能够解决 Chrome 浏览器同步的神器（你们都懂）。

实用工具

集装箱

集装箱是一款包含了四大功能的「超级神器」

1. 访问 Google 服务（搜索，Chrome 应用店、学术、Gmail、同步 Chrome 浏览器等）

2. 网盘助手&网盘链接检查：自动识别网盘链接，如果提取码被分享，即可自动填充

3. 优惠购：支持查看主流电商平台的商品历史价格，以及查询和获取相关的优惠券

4. 下载卫士：在华军、西西、下载吧等各大主流下载站，隐藏所谓的「高速下载」选项


长截图（Full Page Screen Capture）

长截图在手机上是很普及的功能，但电脑上还很少见到，Full Page Screen Capture 就是一款实用的长截图工具，它可以一键截取你想要保存的网页和长文章，同时它还支持把长截图另存为 PDF 文档或者 JPG、PNG 图片格式。


新标签页

MONKNOW

提到 Chrome 上的新标签页，可能很多人都知道 Infinity Pro，但其实还有一款设计更精美，功能更丰富、自由度更高，同时还支持暗色模式的新标签页——MONKNOW。


MONKNOW 的首页定制化程度很高，就像安卓手机上的「抽屉」功能一样，它不仅能添加很多网站，还可以根据类型建立分组，而且 MONKNOW 针对大部分网站 ICON 都做了重新设计，风格就是图中的圆角矩形大色块，色彩十分鲜明，另外还有四种不同的主视图样式以供选择。


MONKNOW 还有很多贴心的小设计，大家可以自行安装体验，同时它的云同步功能肯定没有落下，无论你身在何方，只要安装了 MONKNOW，这就是一个只属于你的浏览器。

视频下载

IDM

IDM 相信大家都不陌生，这是一款「无所不能下载」的软件。

安装 IDM 客户端后，把它的扩展程序关联至 Chrome 浏览器，它就可以自动嗅探网页上视频资源，在视频右上角的浮窗菜单中选定所需要的视频资源就可以下载了。另外除了视频，其他的应用程序、图片、压缩包等资源也可以嗅探到，十分方便快捷。

不过在这里多说一句，下载使用时一定要注意版权问题，尤其是涉及商用的，要谨慎处理。


效率工具

清爽版百度（需要 Tampermonkey 支持）

安装 Tampermonkey 扩展后，继续安装 AC-baidu 脚本即可激活此功能。

无论是百度首页还是搜索页面，总是充斥着无聊的热搜、狗皮膏药般的小广告，还有各种骗点击的推广信息，别烦，AC-baidu 就是为了干掉这些而生的。


开启 AC-baidu 后，可以看到百度首页除了搜索框，其他的资讯、热搜都已经不复存在。


而在搜索结果页，AC-baidu 不仅可以过滤掉无用的信息，还可以把搜索结果以卡片的样式呈现出来，看起来也更加直观。如果你使用的是宽屏显示器，也可以设置成双列或者三列、甚至四列布局，以获得更高的浏览效率，另外除了百度，这些功能还支持 Google、搜狗和必应搜索引擎。


视频速度控制器（Video Speed Controller）

尽管视频网站已经提供了倍速观看的功能，但依然有很多小伙伴们不满足于最高 2 倍速的设置，Video Speed Controller 就是一款能够自由控制视频播放的扩展，它可以内嵌到网页播放器中（可惜不支持百度云），最高支持 16 倍速播放，也可以通过键盘上的 S 和 D 键快捷调整播放速度，当然你也可以指定其他的快捷键。


除了调整播放速度，它还有快进和快退的功能，并且快进和快退的时间也是可以调整的，默认为 10 秒。


找回关闭页面（SimpleUndoClose）

浏览网页时，免不了会手滑关掉一个重要的页面，或者是突然需要查找数小时前的某个网页，这时候如果去历史记录里漫无目的寻找，无异于大海捞针，真的是心态要崩了……不过别担心，SimpleUndoClose 就是为了解决这个问题而生的，安装后只需点一下扩展面板的小箭头，就可以罗列出过去关掉的所有标签页。如果内容太杂乱，下方的搜索框里还可以精准查找。


扩展管理器

Chrome 之所以强大，与它丰富的扩展支持是离不开的，但 Chrome 本来就是个「吃内存大户」，安装这么多扩展后，对系统内存更是雪上加霜，搞不好还会造成浏览网页卡顿。好在扩展管理器可以对这些进行管理，在它的控制面板中，可以选择关闭一些使用频次低的扩展，需要的时候再开启，比如长截图，这样可以节省很多系统资源，以免造成不必要的浪费。


这是 Chrome 扩展推荐的第四期，之后我们会介绍更多强大、实用的 Chrome 扩展程序，如果你也知道一些效率神器，欢迎在评论区里分享出来哦。
